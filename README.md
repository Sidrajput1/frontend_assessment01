# Frontend Assessment Submission

## Overview

This project is a React-based frontend implementation created from the provided Figma design. The focus was on building reusable components, responsive layouts, and dynamic UI rendering using React hooks.

## Tech Stack

- React
- Vite
- CSS
- Local JSON Data

## Features

- Responsive card-based layouts
- Interactive FAQ accordion
- Dynamic rendering using local JSON files
- Loading and error state handling
- Reusable React components
- Smooth hover animations and transitions

## Sections Implemented

1. Why Whistle?
2. Results You'll Love
3. FAQ Accordion

## How to Run

```bash
npm install
npm run dev
```

## Approach

I focused on recreating selected sections from the Figma design as closely as possible while keeping the code modular and reusable. Local JSON files were used to simulate API-driven content, and React hooks were used for state management and interaction handling.
